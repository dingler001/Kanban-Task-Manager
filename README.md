# Kanban Task Manager

A simple Kanban-style task management application built with **React**.

## Live Demo
https://kanban-task-manager-rust.vercel.app/
## Features


* Add tasks
* Move tasks across workflow stages
* Edit tasks
* Delete tasks
* Drag and drop tasks between coloumns
* Persistent storage with localStorage
* Modern UI with galaxy background
